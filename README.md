These are the the lecture slides for a one-semester course on deep reinforcement learning, held at [TU Dortmund University](https://www.tu-dortmund.de/) during the Summer term 2026, organized by the [Safe Autonomous Systems](https://sas.cs.tu-dortmund.de/) group (Department of Computer Science). The slides have been created using the presentation software [Decker](https://github.com/decker-edu/decker/tree/master).

The slides can be found under [https://sebastianpeitz.github.io/DeepRL_Course](https://sebastianpeitz.github.io/DeepRL_Course/).
