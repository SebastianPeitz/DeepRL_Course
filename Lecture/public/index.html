How to use Decker Anleitung für Decker

Navigation und Interaktion

Die Oberfläche eines Foliensatzes ermöglicht folgende Aktionen:

  • Mit den Tasten Cursor-links und Cursor-rechts kann man durch die Folien blättern.
  • Mit den Tasten Cursor-runter und Cursor-hoch kann man — falls vorhanden — Unterfolien zur aktuellen Folie aufrufen.
  • Die Taste f schaltet in den Vollbildmodus. Dieser kann mit der Taste Esc verlassen werden.
  • Mit der Taste ? wird eine Liste aller Tastenkürzel angezeigt.
  • Der Burger-Menu-Button (links oben) öffnet das Navigationspanel.
  • Der Fragezeichen-Button (rechts oben) öffnet das Feedback-Panel, wo zur aktuellen Folie (anonym) Fragen gestellt und Kommentare gegeben werden können.
  • Der Play-Button (rechts oben) startet — wenn vorhanden — die Videoaufzeichnung bei der aktuellen Folie.
  • Einzelne Elemente (z.B. Bilder) können per Doppel-, Dreifach- oder Alt-Klick herangezoomt werden (je nach Konfiguration).
  • Bei Folien mit angehängten Whiteboard-Seiten blinkt der untere Rand beim Folienwechsel. Runter-scrollen, um diese Inhalte zu sehen.
  • Eingebettete Demos können mit dem Expandieren-Button (rechts oben) in den Vollbildmodus geschaltet werden.

Navigationspanel

Im Navigationspanel können folgende Aktionen ausführt werden:

  • Zu einer beliebigen Folie springen.
  • Der Home-Button verlässt den Foliensatz und navigiert zurück zur Übersichtsseite.
  • Der Such-Button öffnet einen Suchdialog (Tastaturkürzel: Strg-f oder Cmd-f).
  • Der Drucken-Button schaltet in die Druck-Ansicht, wo die Präsentation als PDF exportiert werden kann.
  • Die Hell/Dunkel-Buttons und wechseln zwischen Light-Mode und Dark-Mode.
  • Der Handout-Button wechselt zwischen Folien- und Handout-Ansicht (Tastaturkürzel: h-h-h).
  • Der Präsentations-Button schaltet den Präsentationsmodus an/aus (Tastaturkürzel: p-p-p).
  • Der Aufnahme-Button initialisiert die Videoaufnahme (Tastaturkürzel: r).
  • Der Barrierefreiheit-Button schaltet Kontrastfarben und andere Barrierefreiheitsfunktionalitäten an/aus (Tastaturkürzel: a-a-a).

Feedback-Panel

Im Feedback-Panel können Fragen/Anmerkungen zur aktuellen Folie gestellt/gemacht werden:

  • Fragen/Anmerkungen können unten links eingegeben und mit Shift-Enter oder dem Senden-Button abgeschickt werden.
  • Sie können Fragen, die Sie beantwortet sehen wollen, mit dem Daumen-hoch--Button befürworten.
  • Eigene Fragen können mit dem Bearbeiten-Button bearbeitet werden.
  • Eigene Fragen können mit dem Mülltonnen-Button gelöscht werden.

Videoaufzeichnungen

Beim Abspielen der Videoaufzeichnung können folgende Tastaturkürzel verwenden werden:

  • Esc: Video schließen und zu den Folien zurückkehren.
  • Leertaste/k: Video pausieren.
  • Cursor-links/Cursor-rechts oder j/l: Im Video um 10 Sekunden zurück/vorwärts springen.
  • Page-Up/Page-Down: Im Video zur vorheringen/nächsten Folie springen.
  • Cursor-hoch/Cursor-runter: Lautstärke um 5% erhöhen/verringern.
  • m: Video stummschalten.
  • Auf Touch-Geräten: Doppel-Tippen am linken/rechten Bildschirmrand springt 10 Sekunden zurück/vor.
  • Auf Touch-Geräten: Doppel-Tippen in der Mitte pausiert das Video.

Zusätzliche Hinweise für seh-eingeschränkte Personen

Für Nutzer von Screen-Readern sind folgende Hinweise hilfreich:

  • Das Navigationspanel ist das erste interaktive Element auf den Folien.
  • Die Buttons für vorherige und nächste Folie sind die letzten interaktiven Elemente auf den Folien.
  • Die Seitennummer zwischen den Vor- und Zurück-Buttons ist ein Link ohne weitere Funktion, der verwendet werden kann, um schnell zu den Navigationsbuttons zu springen.
  • Im Navigationspanel sind der Schließen-Button, die Aktions-Buttons und die Folienliste eigene Tab-Gruppen, innerhalb derer mit den Cursor-Tasten navigiert werden kann.
  • Im Feedback-Panel ist die Fragenliste eine eigene Tab-Gruppe:
    • Sie können innerhalb der Fragenliste mit den Cursor-Taste navigieren.
    • Sie können mit einer selektierten Frage mit der Eingabe-Taste interagieren.
    • Sie können eine selektierte Frage mit der Esc-Taste wieder abwählen.
  • Im Suchdialog können Sie mit Eingabe durch die Folien navigieren, in denen der Such-Begriff gefunden wurde.
  • Matheformeln werden mit MathJax dargestellt und können mit Tab selektiert werden:
    • Leertaste öffnet ein Menu zum Einstellen der assistiven Darstellung der Formeln.
    • Eingabe öffnet einen Untersuchungsdialog, in dem die Formel mit den Cursor-Tasten näher untersucht werden kann.

Grundsätzlich ist die Betrachtung der Folien im Handout-Modus zu empfehlen. Im Präsentationsmodus ist nur die aktuelle Folie von assistiver Technologie untersuchbar. Im Handout-Modus ist die gesamte Präsentation für assistive Technologie untersuchbar. Insbesondere können Sie die Überschriften der Folien als Sprungmarken verwenden.

Navigation and Interaction

The basic slide interface allows the following actions:

  • The keys cursor-left and cursor-right can be used to navigate through the slides.
  • The keys cursor-up and cursor-down can be used to navigate to sub-slides, if available.
  • The key f activates fullscreen mode. You can leave it with key Esc.
  • The key ? shows a cheat sheet listing all keyboard shortcuts.
  • The burger-menu button (top left) opens the navigation panel.
  • The question-mark button (top right) opens the feedback panel, where you can (anonymously) ask questions and give feedback about the current slide.
  • The play button (top right) starts the video recording at the current slide (if available).
  • You can zoom in on each element by double-, triple-, or alt-clicking (depending on the current configuration).
  • A slide with additional whiteboard pages will have a blinking bottom border. Scroll down on these slides to see the whiteboard annotations.
  • Embedded demos and websites can be expanded to fullscreen mode with the expand button (top right).

Navigation Panel

The navigation menu contains the following features:

  • Jump to any slide by selecting it in the list.
  • The home button leaves the slide deck and navigates to the index page.
  • The search button opens the search dialog (keyboard shortcut Ctrl-f or Cmd-f).
  • The print button opens the print view, from where you can export the slides as PDF.
  • The light/dark buttons and switch between light mode and dark mode.
  • The handout button switches between handout mode and slide mode (keyboard shortcut h-h-h).
  • The presenter mode button toggles the presenter mode (keyboard shortcut p-p-p).
  • The recording button initialize the video recording (keyboard shortcut r).
  • The accessibility button toggles contrast colors and other accessibility features (keyboard shortcut a-a-a).

Feedback Panel

The feedback panel can be used to post questions and comments for the current slide:

  • Enter question/comment in the input form and submit it with either Shift-Enter or the send button .
  • Up-vote questions that you want to see answered with the thumbs-up button .
  • Edit your own question with the edit button .
  • Delete your own question with the trashcan button .

Playback of Video Recordings

When you are watching the embedded video recording you can use the following controls:

  • Key Esc closes the video and navigates to the slide currently visible in the video.
  • Keys Space/k pauses the video.
  • Keys Cursor-left/Cursor-right or j/l jump backward/forward by ten seconds.
  • Keys Page-Up/Page-Down jump backward/forward by one slide.
  • Keys Cursor-up/Cursor-down increase/decrease audio volume.
  • Key m: Mute/unmute.
  • On touch devices: Double tap at the left/right screen boundary jumps backward/forward by ten seconds.
  • On touch devices: Double tap at the center of the screen pauses the video.

Additional information for visually impaired users

The following hints may be useful for users of screen readers:

  • The first element in tab-order is the navigation menu.
  • The last elements in tab-order are the buttons to navigate to the previous or next slide.
  • The slide number between the previous and next buttons is a link without further function, so it can be used to quickly navigate to these buttons.
  • The navigation menu has three tab-groups: The close button, the action buttons and the slide list. Within these groups you can select different elements with the cursor keys.
  • In the feedback panel, if there are questions available, you can tab-navigate into the list of questions and navigate within the questions with the following keys:
    • Navigate between questions with the cursor keys.
    • Interact with a question with the Enter key.
    • Return to the question list with the Esc key.
  • Within the search dialog you can press Enter to jump to the slides the entered keyword was found.
  • Math formulas are rendered using the MathJax library. You can interact with them with the following keys when the accessibility features are enabled:
    • Space opens a menu to change the settings of your assistive view.
    • Enter opens a dialog that can help you navigate through complex formulas and equations with the cursor keys.

For using a screen reader we recommend navigating the slides in handout mode. The usual presentation view allows assistive technology to see the current slide only, while the handout mode allows assistive technology to see the entire contents of the presentation. You can then in particular use the slide headings for navigation.
